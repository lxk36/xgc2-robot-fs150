# FS150 PX4

This directory is for real FS150 PX4 resources and notes.

Expected content:

- exported firmware and parameter provenance notes;
- custom airframe and board observations;
- serial/MAVLink port mapping;
- sensor and actuator records that describe the real aircraft.

Do not put SITL-only launch files or Gazebo models here.  Those belong in the
FS150 SITL repository.
